## [stitches](https://github.com/stitchfix/stitches)

### Create a Microservice in Rails with minimal ceremony


Add to your `Gemfile`:

```ruby
gem 'stitches'
```

Then:

```
bundle install
```

Then, set it up:

```
> bin/rails generate stitches:api
> bundle exec rake db:migrate
```
