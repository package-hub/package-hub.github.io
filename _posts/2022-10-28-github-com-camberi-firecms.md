## [firecms](https://github.com/Camberi/firecms)

### Awesome Firebase/Firestore-based CMS. The missing admin panel for your Firebase project!


FireCMS is based on this great technologies:

- Typescript
- Firebase SDK 9
- React + React Router 6
- Material UI 5
- Formik + Yup
