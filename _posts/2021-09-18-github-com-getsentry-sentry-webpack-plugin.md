## [sentry-webpack-plugin](https://github.com/getsentry/sentry-webpack-plugin)

### Official webpack plugin for Sentry https://sentry.io


Using npm:

```bash
$ npm install @sentry/webpack-plugin --save-dev
```

Using yarn:

```bash
$ yarn add @sentry/webpack-plugin --dev
```
