## [mongo-ruby-driver](https://github.com/mongodb/mongo-ruby-driver)

### The Official MongoDB Ruby Driver


Install via RubyGems, either via the command-line for ad-hoc uses:

  $ gem install mongo

Or via a Gemfile for more general use:

  gem 'mongo'
