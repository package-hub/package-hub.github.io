## [Specify](https://github.com/Codeception/Specify)

### BDD style code blocks for PHPUnit / Codeception


*Requires PHP >= 7.4*

* Install with Composer:

```
composer require codeception/specify --dev
```

* Include `Codeception\Specify` trait in your tests.
