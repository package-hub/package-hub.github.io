## [napa](https://github.com/bellycard/napa)

### A simple framework for building APIs with Grape


Napa is available as a gem, to install it run:

```
gem install napa
```

Or, if you're using Bundler, add it to your Gemfile:

```
gem 'napa'
```

And run:

```
$ bundle install
```
