## [localer](https://github.com/aderyabin/localer)

### Automatic detecting missing I18n translations tool.


Add this line to your application's Gemfile:

```ruby
gem 'localer'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install localer
