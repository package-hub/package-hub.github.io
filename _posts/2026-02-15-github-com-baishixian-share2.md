## [Share2](https://github.com/baishixian/Share2)

### Content Share For Android.


Share2 利用了 Android 系统的原生 API 实现了分享功能，支持文本信息、图片、音视频等其他类型文件的分享，兼容处理了不同系统版本下的常见问题，并在一些主流应用上进行了兼容测试。

| 系统兼容 | 版本号 |
|:----:|------|
| 最低 | API 11 |
| 最高 | API 27 |

> 如果发现 Share2 存在不兼容的情况，请在 Issues 中提出。
