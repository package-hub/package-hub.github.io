## [react-monaco-editor](https://github.com/react-monaco-editor/react-monaco-editor)

### Monaco Editor for React.


To build the examples locally, run:

```bash
yarn
cd example
yarn
yarn start
```

Then open `http://localhost:8886` in a browser.
