## [fhir_client](https://github.com/fhir-crucible/fhir_client)

### Ruby FHIR Client


Ruby FHIR client.

Supports:
* FHIR R4, STU3 and DSTU2
* JSON and XML
* All CRUD, including version read and history
* Transactions and Batches
* Search
* Operations (e.g. `$everything`, `$validate`)
* Support for OAuth2
